# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch. Patch releases are not maintained
for historical snapshots.

## Reporting a Vulnerability

If you discover a security issue, please report it privately.

- Email: mystic-security@example.com
- Include steps to reproduce and any relevant logs.

We will acknowledge reports within 72 hours and provide a remediation plan or
fix timeline as soon as possible.

## Public Repository Hygiene

The public repository must not include:

- Secrets, API keys, or credentials
- Production endpoints or private infrastructure references
- Sensitive data or customer identifiers

If you notice any of the above, report it immediately.
